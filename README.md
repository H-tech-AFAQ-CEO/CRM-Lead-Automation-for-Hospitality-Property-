# CRM Lead Automation for Hospitality Property

A comprehensive CRM (Customer Relationship Management) system designed specifically for automating lead management and customer interactions in the hospitality property industry.

## Overview

This project provides an integrated solution for hospitality properties to manage leads, automate follow-ups, and streamline customer relationship processes. It combines Python backend logic with an intuitive HTML-based frontend interface.

## Tech Stack

- **Frontend**: HTML (80.5%)
- **Backend**: Python (19.5%)

## Features

- **Lead Management**: Track and organize leads from various sources
- **Automation**: Automated workflows for lead follow-ups and customer engagement
- **Hospitality-Specific**: Tailored for hotels, resorts, and property management
- **User-Friendly Interface**: Intuitive web-based dashboard

## Getting Started

### Prerequisites

- Python 3.x
- Web browser (for HTML interface)
- Basic understanding of CRM concepts

### Installation

1. Clone the repository:
```bash
git clone https://github.com/H-tech-AFAQ-CEO/CRM-Lead-Automation-for-Hospitality-Property-.git
```

2. Navigate to the project directory:
```bash
cd CRM-Lead-Automation-for-Hospitality-Property-
```

3. Install Python dependencies:
```bash
pip install -r requirements.txt
```

### Usage

1. Start the backend service (Python)
2. Open the HTML files in your web browser to access the frontend interface
3. Configure your property details and lead sources
4. Set up automated workflows and follow-up schedules
